## 🏆 功能
1. 简约记账，数据保存在云端，支持多设备登录；

2. 支持添加账本、可以和好友共享账本内容；

3. 支持微信、支付宝等账单导入，免去每日记账烦恼；

4. 支持自定义分类和排序，收支一目了然；

5. 每日一句，开启全新每一天；
